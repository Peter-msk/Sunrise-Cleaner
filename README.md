# Sunrise Cleaner

> Windows의 임시파일과 캐시를 사용자가 직접 확인·선택한 뒤 안전하게 정리하는 시스템 정리 도구

## 1. 프로그램 소개

Sunrise Cleaner는 Windows PC에 누적되는 임시파일, 캐시, 휴지통 항목과 일부 시스템 정리 대상을 찾아 사용자가 검토한 뒤 삭제할 수 있도록 만든 프로그램입니다.

무조건 많은 파일을 지우는 방식이 아니라, 삭제 후보를 먼저 탐색하고 종류·경로·예상 용량·권한 필요 여부를 보여준 다음 사용자가 선택하고 최종 승인한 항목만 정리하는 것을 핵심 원칙으로 합니다.

일반 권한에서 처리할 수 있는 항목을 먼저 제공하고, Windows 보호영역이나 서비스 제어가 필요한 항목은 별도의 관리자 모드에서만 선택할 수 있도록 구분합니다.

## 2. 개발 목적

1. 반복적으로 쌓이는 불필요한 임시파일과 캐시를 쉽게 확인하고 정리합니다.
2. 삭제 전에 대상과 용량을 투명하게 표시하여 사용자가 직접 판단하게 합니다.
3. 일반 권한 우선 실행으로 불필요한 관리자 권한 요구를 줄입니다.
4. 대량 파일과 느린 디스크에서도 프로그램이 멈춘 것처럼 보이지 않도록 진행 상태를 유지합니다.
5. 실행 중인 프로그램과 사용자 지속 데이터를 보호합니다.
6. 삭제와 업데이트처럼 보안 제품이 민감하게 보는 기능을 단순하고 검증 가능한 구조로 운영합니다.

## 3. 주요 기능

### 정리 대상 탐색

- 현재 사용자 임시폴더
- Windows에서 생성되는 안전한 임시파일과 로그 후보
- 지원 브라우저의 캐시 후보
- Windows 썸네일·일부 사용자 캐시
- 사용자 폰트 캐시
- Microsoft Store/UWP 앱의 오래된 TempState·LocalCache Temp·AC Temp 하위 항목
- Windows 휴지통
- DNS Resolver Cache
- 관리자 권한이 필요한 Windows Update 다운로드 캐시
- Delivery Optimization 캐시
- Print Spooler 대기열 파일
- 서비스 프로필과 Windows/ProgramData 보호영역의 승인된 후보

실제 지원 항목은 프로그램 버전과 Windows 환경에 따라 달라질 수 있으며, 프로그램은 각 후보의 정책과 권한을 다시 확인한 뒤 표시합니다.

### 안전한 후보 선택

- 삭제 전 후보 목록 표시
- 후보 종류, 경로, 예상 또는 실제 용량 표시
- 일반 권한 후보와 관리자 권한 후보 구분
- 사용자가 선택한 항목만 삭제
- 최종 확인 없는 삭제 금지
- 관리자 권한이 필요한 후보는 일반 모드에서 잠금
- 동등한 대량 후보를 안전한 그룹으로 묶어 선택 창의 과부하 감소
- 그룹 삭제 직전에 실제 하위 경로를 다시 확장하고 개별 재검증

### 대량 파일 처리

- 디렉터리 용량 계산 워커 수 제한으로 디스크 경합 감소
- 이미 계산한 정확한 용량 재사용
- 대량 폴더의 제한시간·파일 수 상한과 추정값 표시
- 수천 개 후보를 한 번에 UI에 넣지 않고 분할 로딩
- 검색·용량 계산·삭제 진행 이벤트 병합으로 UI 응답성 유지
- 대용량 폴더를 작은 단위로 나누어 처리하는 협력형 단일패스 삭제
- 현재 처리 단계와 heartbeat 표시
- 후보별·단계별 처리 시간 분석 기록

### 보호 정책

- `Packages` 전체 또는 UWP 앱 루트 삭제 금지
- `LocalState`, `RoamingState`, 설정 DB 등 지속 데이터 제외
- 심볼릭 링크와 reparse point를 따라가며 삭제하지 않음
- 실행 중인 브라우저나 관련 프로그램의 캐시 후보 보호
- Windows 핵심 보호경로 차단
- 잠긴 파일과 접근 불가 항목을 실패 또는 건너뜀으로 구분
- 서비스 중지가 필요한 정리는 작업 후 서비스 복구 상태 확인
- 삭제 결과와 건너뛴 이유를 로그로 기록

### 사용자 편의

- 일반 모드와 관리자 모드 분리
- 검색 진행률과 삭제 진행상태 표시
- 예상 확보 용량 확인
- 정리 결과 요약
- 공지사항과 업데이트 확인
- 결과와 오류 진단을 위한 로그
- 단일 실행파일 기반의 간단한 배포

## 4. 기본 사용 흐름

```text
일반 모드 또는 관리자 모드 선택
→ 정리 후보 검색
→ 후보 종류·경로·용량·권한 확인
→ 정리할 항목 선택
→ 최종 승인
→ 삭제 진행상태 확인
→ 삭제·건너뜀·실패 결과 확인
```

## 5. Sunrise Cleaner가 하지 않는 일

- 사용자 승인 없이 자동으로 파일을 삭제하지 않습니다.
- 모든 파일을 위험하게 일괄 삭제하지 않습니다.
- 레지스트리를 무차별적으로 정리하거나 최적화하지 않습니다.
- Windows 핵심 파일과 앱의 지속 데이터를 정리 대상으로 취급하지 않습니다.
- 시작과 동시에 관리자 권한을 강제로 요구하지 않습니다.
- 원격 스크립트를 다운로드하여 실행하지 않습니다.
- 시작프로그램, 서비스, 예약 작업을 몰래 등록하지 않습니다.
- 사용자 파일과 삭제 목록을 외부 서버로 자동 전송하지 않습니다.

## 6. 개발 방향

- Windows 버전별 안전 후보 정책과 보호경로 데이터 정교화
- 브라우저와 UWP 앱의 캐시 구조 변화에 대한 호환성 강화
- 대량 후보 탐색과 용량 계산 속도 개선
- 느린 HDD, 외장 디스크, V3 실시간 검사 환경에서의 응답성 향상
- 실행 중 앱과 잠긴 파일 판별 정확도 개선
- 삭제 전 위험도와 예상 효과를 더 쉽게 이해할 수 있는 UI 제공
- 관리자 서비스 제어형 정리의 복구 확인과 장애 진단 강화
- 정상 삭제 후 남은 빈 폴더·임시파일 처리 정책 개선
- 삭제 결과 통계와 장기적인 공간 사용 변화 확인 기능 검토
- 단일 EXE 업데이트의 다운로드, SHA-256 확인, 교체, 재실행, 잔여 백업 정리 안정화
- AhnLab/V3 오탐지 최소화를 위한 no-UPX, 제품 등록정보, 코드서명, 릴리즈 검증 유지

## 7. 지원 환경과 배포

- 대상 운영체제: Windows 10/11 64비트
- 기본 배포 형태: PyInstaller onefile 단일 EXE
- 현재 공개 README 작성 기준: Sunrise Cleaner v2.9.1
- 공식 실행파일명: `Sunrise_cleaner.exe`
- 일부 시스템 정리 항목은 Windows 관리자 권한이 필요합니다.

## 프로젝트 공통 원칙

Sunrise Group의 프로그램은 단순한 일회성 스크립트가 아니라 실제 업무에서 반복 사용할 수 있는 Windows 데스크톱 도구를 목표로 합니다.

- 원본 파일과 사용자 데이터를 우선적으로 보호합니다.
- 삭제, 다운로드, 업데이트, 외부 프로그램 실행 등 중요한 동작은 사용자가 인지하고 승인한 경우에만 수행합니다.
- 사용자 문서, 이미지, 프로젝트 데이터와 API 키를 불필요하게 외부로 전송하지 않습니다.
- 장시간 작업은 진행 상태와 현재 처리 단계를 표시하고, 오류 발생 시 사용자용 안내와 진단 로그를 분리합니다.
- 공식 배포본은 제품 정보, SHA-256 무결성 확인, 코드 서명, AhnLab/V3 사전점검 등 Sunrise Group의 배포 기준을 따릅니다.
- 기능 추가보다 기존 기능의 재현성, 복구 가능성, 대용량 처리 안정성, 유지보수성을 우선합니다.
