Sunrise Cleaner 공지사항

현재 최신 버전: v2.6.3

- 업데이트 배포 파일 생성 절차를 개선했습니다.
- 빌드 후 실행파일의 무결성 값을 자동 계산하여 latest.json에 반영합니다.
- GitHub Pages에 올릴 latest.json과 notice.txt가 실행파일과 같은 dist 폴더에 생성됩니다.
- 업데이트 안내 화면은 사용자에게 필요한 변경사항과 안정화 내용 중심으로 표시됩니다.
- Sunrise Cleaner는 설치파일이 아닌 단일 실행파일 배포 구조를 유지합니다.
- 업데이트는 사용자가 직접 승인한 경우에만 진행됩니다.
- 정리 작업은 기존처럼 검색/검토 후 사용자가 선택한 항목만 삭제합니다.
