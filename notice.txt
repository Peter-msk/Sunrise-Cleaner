Sunrise Cleaner 공지사항

현재 최신 버전: v2.6.1

- Sunrise Cleaner는 설치파일이 아닌 단일 실행파일 EXE 배포 구조를 유지합니다.
- Sunrise Capture의 보안 및 업데이트 방향을 Sunrise Cleaner 단일 EXE 구조에 맞게 반영했습니다.
- 업데이트 정보는 GitHub Pages latest.json에서 확인하고, 공지사항은 notice.txt 본문을 표시합니다.
- 실제 배포 파일은 GitHub Releases의 Sunrise_cleaner.exe asset을 기준으로 합니다.
- 자동 다운로드, 자동 실행, 자동 설치는 하지 않습니다.
- 사용자가 [업데이트 확인] 버튼에서 직접 승인한 경우에만 새 EXE 다운로드 및 교체 절차가 진행됩니다.
- 새 EXE는 기존 실행파일과 같은 폴더에 임시 다운로드된 뒤, 현재 프로그램 종료 후 기존 위치로 교체됩니다.
- sha256 값은 공식 배포 시 필수 입력 기준입니다. SHA256 검증에 실패하면 업데이트를 중단합니다.
- 현재 v2.6.1은 signtool.exe 없이 배포 가능한 구조입니다.
- Authenticode 공식 코드서명은 Sunrise Group 사업자 등록 및 공인 코드서명 인증서 발급 후 적용 예정입니다.
- PowerShell 런타임 실행과 Registry 런타임 쓰기는 사용하지 않습니다.
