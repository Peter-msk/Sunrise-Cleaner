Sunrise Cleaner 공지사항

현재 최신 버전: v2.8.0

- 삭제 후보 용량 계산을 디스크 부하 제한형 I/O 워커 구조로 개선했습니다.
- 이미 계산된 크기는 재사용하고 파일 크기는 즉시 확인하여 중복 순회를 줄였습니다.
- 계산 중 진행 상태를 계속 표시하며 후보별·전체 계산 제한으로 장시간 정체를 방지합니다.
- Microsoft Store/UWP 앱의 오래된 안전 임시영역 후보를 추가했습니다.
- 현재 사용자 휴지통, DNS 캐시, 사용자 폰트 캐시는 일반 권한에서 선택할 수 있습니다.
- Windows Update, 인쇄 대기열, 서비스 프로필 및 Windows 보호영역은 관리자 모드를 유지합니다.
- Prefetch, WinSxS, Installer, Windows.old, SoftwareDistribution 전체 삭제 제외 정책은 유지됩니다.
- 정리 작업은 검색/검토 후 사용자가 선택한 항목만 삭제합니다.
- 업데이트는 사용자가 직접 승인한 경우에만 진행됩니다.
