Sunrise Cleaner 공지사항

현재 최신 버전: v2.6.8

- 업데이트 확인 버튼과 공지사항 보기 버튼을 분리했습니다.
- 업데이트 확인에서 사용자가 아니요를 선택해도 공지사항 창이 자동으로 뜨지 않도록 수정했습니다.
- 공지사항은 사용자가 [공지사항 보기]를 직접 누른 경우에만 표시됩니다.
- v2.6.7의 삭제 후보 목록 체크 표시 확대와 삭제 가능 항목 전체 선택 기능은 유지됩니다.
- 브라우저 캐시, 폰트 캐시, DNS 캐시, 휴지통 비우기 등 수동 후보 정책은 유지됩니다.
- 업데이트 임시 실행파일이 실행파일 위치에 남지 않도록 처리한 v2.6.6 기준도 유지됩니다.
- 정리 작업은 기존처럼 검색/검토 후 사용자가 선택한 항목만 삭제합니다.
- 업데이트는 사용자가 직접 승인한 경우에만 진행됩니다.
