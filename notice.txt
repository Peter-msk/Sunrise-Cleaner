Sunrise Cleaner 공지사항

현재 최신 버전: v2.7.4

- 삭제 단계의 실시간 로그를 시작/진행/완료 요약 중심으로 간단하게 줄였습니다.
- 삭제 실패와 삭제 제외 항목은 실시간 로그에 계속 표시합니다.
- 상세 삭제 내역은 삭제 결과 목록과 리포트에 유지합니다.
- 실시간 로그 창의 최대 줄 수를 제한해 장시간 작업 중 UI 부담을 줄였습니다.
- 대용량 캐시 삭제의 고속 삭제/정밀 재시도 구조는 유지합니다.
- 검토/삭제 경로명이 길어져도 실행 제어 영역이 흔들리지 않는 고정 레이아웃을 유지합니다.
- Prefetch, WinSxS, Installer, Windows.old, SoftwareDistribution 전체 삭제 제외 정책은 유지됩니다.
- 정리 작업은 기존처럼 검색/검토 후 사용자가 선택한 항목만 삭제합니다.
- 업데이트는 사용자가 직접 승인한 경우에만 진행됩니다.
